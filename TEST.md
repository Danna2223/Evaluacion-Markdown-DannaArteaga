# TEST Markdown

## ¿Qué es un archivo Markdown y para qué sirve?
Un archivo Markdown es un archivo de texto que usa una sintaxis sencilla para dar formato al contenido, como títulos, listas o enlaces. Se utiliza principalmente para documentar proyectos en repositorios.
## ¿Cuál es el propósito del archivo README.md en un repositorio?
*El archivo README.md sirve para explicar de qué trata el proyecto, cómo usarlo y dar información importante.*
## Sintaxis de Markdown

| Elemento | Cómo se escribe |
|--------|----------------|
| Título H1 | # Título grande |
| Título H2 | ## Título mediano |
| Título H3 | ### Título pequeño |
| Lista ordenada | 1. Elemento lista con números |
| Lista desordenada | - Elemento lista con puntos |
| Cursiva | *texto* |
| Negrita | **texto** |
| Enlace | [Google](https://www.google.com) |
